# Лицей для малышей — Детский образовательный центр

Сайт детского образовательного центра для детей от 1 до 7 лет.

## 📁 Структура проекта

project/
├── index.html          # Главная страница
├── css/
│   └── styles.css      # Все стили
├── js/
│   ├── icons.js        # Библиотека SVG-иконок
│   ├── data.js         # Конфигурация сайта (тексты, программы, фото)
│   └── app.js          # Основной JavaScript
├── assets/             # Логотипы и другие изображения
└── photos/             # Фотографии галереи


## 🛠️ Как редактировать

### Тексты и контент
Все тексты, программы, контакты и настройки находятся в `js/data.js`:

```javascript
// Пример: измените телефон или программы
siteConfig.header.phone = "8 (928) 28-28-108";
siteConfig.programs = [ ... ];
```

### Иконки
Добавляйте иконки в `js/icons.js`:

```javascript
'my-icon': '<svg viewBox="0 0 24 24">...</svg>'
```

### Стили
Все стили в `css/styles.css`.

## 📱 Адаптивность

- **Desktop** — 4 колонки в сетке
- **Tablet** (960px) — 2 колонки
- **Mobile** (640px) — 1 колонка

## 📸 Галерея

Фото добавляются в папку `photos/` и подключаются в `data.js`:

```javascript
gallery: [
  {
    main: "01-main.png",
    sideTop: "01-side-top.png",
    sideBottom: "01-side-bottom.png"
  }
]
```

## 📄 Лицензия

© 2026 Лицей для малышей. Все права защищены.
```

## 🔧 Настройка

Все настройки в одном файле — `js/data.js`:

| Секция | Описание |
|--------|----------|
| `brand` | Название и логотип |
| `header` / `footer` | Контакты и соцсети |
| `hero` | Заголовки главного блока |
| `heroGridLayout` | Раскладка сетки под разные экраны |
| `infoBlocks` | Карточки (режим работы, адрес, рейтинг) |
| `gallery` | Фотографии для галереи |
| `programs` | Программы обучения |
| `promo` | Промо-блок |

## 🖼️ Добавление фото

1. Положите фото в папку `photos/`
2. В `data.js` добавьте объект в массив `gallery`:

```javascript
gallery: [
  {
    main: "ваше-фото-main.png",
    sideTop: "ваше-фото-top.png",
    sideBottom: "ваше-фото-bottom.png"
  }
]
```
