# 🏫 Лицей для малышей

[![GitHub](https://img.shields.io/badge/GitHub-Repo-blue)](https://github.com/ваш-аккаунт/ваш-репозиторий)

Сайт детского образовательного центра "Лицей для малышей". Разработан с использованием чистого HTML, CSS и JavaScript без фреймворков.

## 📋 Функциональность

- ✅ Адаптивная сетка Hero-блока (Desktop / Tablet / Mobile)
- ✅ Галерея с переключением сцен и лайтбоксом
- ✅ Программы с фильтрацией по возрасту
- ✅ Информационные карточки (режим работы, адрес, рейтинг)
- ✅ Промо-блок
- ✅ Социальные сети в шапке и подвале
- ✅ Телефон в CTA-блоке

## 📦 Технологии

- **HTML5** — семантическая разметка
- **CSS3** — Flexbox, Grid, адаптивность
- **JavaScript (ES5)** — без сборщиков, чистый JS
- **Google Fonts** — шрифт Rubik

## 🔧 Настройка

Все настройки в одном файле — `js/data.js`:

| Секция | Описание |
|--------|----------|
| `brand` | Название и логотип |
| `header` / `footer` | Контакты и соцсети |
| `hero` | Заголовки главного блока |
| `heroGridLayout` | Раскладка сетки под разные экраны |
| `infoBlocks` | Карточки (режим работы, адрес, рейтинг) |
| `gallery` | Фотографии для галереи |
| `programs` | Программы обучения |
| `promo` | Промо-блок |

## 🖼️ Добавление фото

1. Положите фото в папку `photos/`
2. В `data.js` добавьте объект в массив `gallery`:

```javascript
gallery: [
  {
    main: "ваше-фото-main.png",
    sideTop: "ваше-фото-top.png",
    sideBottom: "ваше-фото-bottom.png"
  }
]
```

## 🚀 Развертывание

### GitHub Pages

1. Перейдите в Settings → Pages
2. Выберите ветку `main` и папку `/ (root)`
3. Сайт будет доступен по адресу:
   `https://lecey.github.io/licey/`

### Netlify / Vercel

Просто перетащите папку с проектом или подключите GitHub репозиторий.
